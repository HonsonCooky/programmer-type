Keyboard Inputs

This web app has been designed with context-driven keyboard navigation in mind. Turn off your Vimium and follow the bindings wrapped with braces [X]. Context-driven meaning that if some sub-menu is open, only keys from this menu will be valid next inputs; Anything else will reset the context. You can view the current key combination in the bottom right; Where *base* indicates that you have no context.
- Press Esc to reset the context.
- Press :q to exit the text editor.
