Welcome to programmer-type; A typing test designed by developer for developer.

[JIRA-69420]
As a developer I would like to... test out my sick new keyboard layout on my Advantage 360 with some real world programming tests. I can QWERTY on the MonkeyType, but can I hit Neovim with an anonymous function declaration without breaking a pinky? Here's a quick way to find out.

This web app has been designed with context-driven keyboard navigation in mind. Turn off your Vimium and follow the bindings wrapped with braces [X]. Context-Driven meaning that if some sub-menu is open, only those keys will be valid next inputs. You can view the current key combination in the bottom right; Where *base* indicates that you have no context.
- Press Esc to reset the context.
- Press Esc three times to exit the text editor.
